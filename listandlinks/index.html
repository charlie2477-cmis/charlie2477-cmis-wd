How to create a webpage!

  1. You create a html and a css
  2. Login into your sftp account or create one
  3. Do your coding that you have to do to the html, when you're done click save.
  4. When you are done, you can look at it on the development server by typing this command python -m SimpleHTTPServer
  5. To login into your sftp account you type sftp username@cmis-hsteacher
  6. You get your html and css from your terminal to your sftp by typing put (filename).
  7. Check that you uploaded the correct files.
  8. When you're done type the commit command which is commit -m "Your commit"
  9. Then when you are done with your commit, push the file to your repository.
  10. Type in your username and password and check your repository to make sure you have the file.
